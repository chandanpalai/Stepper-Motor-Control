Detailed Description

Register component.

Author
Marc Kossmann
Michael Riedel
Version
v2.0.0
Date
05.12.2014

Provides the ctrlReg, speedReg, stepsReg registers. The ctrlReg can be set and cleared bitwise with the ctrlSetReg and ctrlClrReg.

Addresses for register-access:

address register
000 ctrlReg
001 ctrlSetReg
010 ctrlClrReg
011 speedReg
100 stepsReg
