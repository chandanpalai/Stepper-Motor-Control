divider integer := 125000
Generic

Prescaler for PWM-signal.

For this purpose 2,5 ms are used as minimal pulse-width.

The prescaler is calculated with the given and desired frequency via the following formula: prescaler = f_clock [Hz] / f_prescaler [Hz] e.g.: f_prescaler = 1/5 ms = 400 Hz prescaler = 50 Mhz / 400 Hz = 125000
